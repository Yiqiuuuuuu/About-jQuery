Notes:

  • $(selector).stop(stopAll,goToEnd);

Explanations:

  • The stop() method works for all jQuery effect functions,
  • stopAll parameter: specifies whether also the animation queue should be cleared or not. Default is false, which means that only the active animation will be stopped, allowing any queued animations to be performed afterwards.
  • goToEnd parameter: specifies whether or not to complete the current animation immediately; default is false.
