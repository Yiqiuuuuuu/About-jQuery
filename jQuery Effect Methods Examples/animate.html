Notes:

  • $(selector).animate({params},speed,callback);

Explanations:

  • required params parameter: defines the CSS properties to be animated.
  • speed parameter: specifies the duration of the effect (values: "slow", "fast", or milliseconds).
  • callback parameter: a function to be executed after the animation completes.
  • By default, all HTML elements have a static position, and cannot be moved.To manipulate the position, remember to first set the CSS position property of the element to relative, fixed, or absolute.
  • animate() can manipulate ALL CSS properties-there is one important thing to remember: all property names must be camel-cased when used with the animate() method; you will need to write paddingLeft instead of padding-left, marginRight instead of margin-right, and so on.
  • Also, color animation is not included in the core jQuery library.If you want to animate color, you need to download the Color Animations plugin from jQuery.com.
