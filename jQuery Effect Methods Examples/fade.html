Notes

  • $(selector).fadeIn(speed,callback);
  • $(selector).fadeOut(speed,callback);
  • $(selector).fadeToggle(speed,callback);
  • $(selector).fadeTo(speed,opacity,callback);

Explanation

  • required speed parameter: specifies the duration of the effect (values: "slow", "fast", or milliseconds).
  • requiredopacity parameter in the fadeTo() method: specifies fading to a given opacity (value between 0 and 1).
  • back parameter: a function to be executed after the function completes.
